Spacing Utility

Now moving onto spacing, this mainly includes stuff like the margin and padding, there's a lot to this it seems, there's a bunch of letters and stuff for different options. Something to make this easier to understand is knowing this formula {property}{sides}-{size} using this formula it's default setting is set to the xs breakpoint. To use the other breakpoints there is another formula that can be used being; {property}{sides}-{breakpoint}-{size}.

Spacing Sizes

Moving onto the different sizes that goes with the spacing properties, there is basically the same amount of these compared to the last section. However, what this does is either removes the margin or padding in some cases or will times the spacer by something. For example "0" would mean that it would set the margin or padding to nothing = removing the specified one. Then with 1 it would times the margin or padding by 0.25, continuing with 2 = 0.5, 3 = 1, 4 = 1.5 and finally 5 = 3.

Display Properties

There's a bunch of useful display properties that can be used, for example making something inline, block, flex etc. But it's also useful for making something have a display of none, making it not show up at all. "d-none" will hide an element, but this can also be done responsively, for example using "d-sm-none" will hide the element on small and above screens.
