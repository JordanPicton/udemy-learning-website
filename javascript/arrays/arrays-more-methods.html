Includes Method

So the Includes method is a boolean, so when using this method on an array, it searches the array for the value inputted and if it's found then it's classed as true, whereas if the value isn't found then it's classed as false.

This makes more sense when used within something like an IF statement due to how it works, you can check IF something INCLUDES something and if it does then it would be classed as true and if not then it would be false.

I've tested this and it works the way I think it does.

This method is very similar to how the IndexOf works with a String. For example if I was to have a String "Hello" and used the IndexOf on that String like so "Hello".indexOf("e") then the output value would be 1 since the index starts at the number 0 and this spot would be taken by the character "H". Searching for a character that's not within that String would give the value of -1 because it's not there.

It seems when using this method on an array that I may have to use the full piece of data rather than just a section of it however, I'll test this out in the script file. This is an easy way to find out if something is within a certain array or not. IF (condition) is equal to -1 then it's not located within this array.

Something else to note about this is that I can start it from a different index if I wanted other than 0, this has nothing to do with the IF statement I was just talking about. But if I wanted to do this then it would look like so arrayName.indexOf("Test", 4); in this case the indexOf would start looking FROM index 4 and then onwards.

Will only show the first instance of the asked for value condition, just like it would for a String. Can also tell if something is in an array or not but can also tell you where the first instance of that value will be shown first.
