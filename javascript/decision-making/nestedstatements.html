So what this is, is basically I can nest conditionals inside conditionals. What this means is that I can basically have an IF statement nested within another IF statement.

With the first example for these nested statements, we'll be asking a user for a password that will have two conditions. One is that the password MUST be 6+ characters and the other is that the password CANNOT include a space/white space character.
